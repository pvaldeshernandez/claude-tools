# Language Rules by Study Design

## Cross-Sectional (non-causal)

When the study design is cross-sectional, use non-causal language throughout.

| Avoid | Use instead |
|-------|------------|
| mediates, mediation | accounts for, explains the association |
| pathway, path | statistical pattern, indirect effect |
| leads to, causes, triggers | co-occurs with, is associated with |
| predicts (ambiguous) | corresponds to, is linked to |
| causal chain | covariance decomposition |
| effect of X on Y | association between X and Y |
| demonstrates that X causes Y | is consistent with X being associated with Y |
| mechanism | statistical pattern, explanatory variable |

When describing mediation results in cross-sectional data:
- "X accounts for the association between A and B" (not "X mediates A→B")
- "The indirect effect through X" (not "the mediation pathway through X")
- "Covariance decomposition consistent with the hypothesized ordering" (not "causal chain")
- In the Discussion, explicitly state: these are cross-sectional associations, not causal inferences

## Longitudinal (temporal ordering established)

When data has repeated measures with temporal ordering:

| Now allowed | Still avoid |
|-------------|------------|
| predicts, prospectively predicts | causes (unless RCT) |
| mediates (with temporal ordering noted) | proves, demonstrates causality |
| precedes, follows | definitive causal claims |
| temporal association | |

## RCT (randomized controlled trial)

Strongest causal language permitted:

| Now allowed | Still avoid |
|-------------|------------|
| causes, leads to | proves definitively, universally |
| treatment effect, causal effect | overgeneralization beyond sample |
| mediates (causal mediation) | |
| intervention produced | |
